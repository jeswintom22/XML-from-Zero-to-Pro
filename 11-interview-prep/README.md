## Interview Themes Covered

- XML fundamentals
- well-formed vs valid XML
- DTD vs XSD
- namespaces
- XPath and XQuery
- XSLT
- XML modeling
- parsing and validation concepts
- BaseX workflows

## Answering Strategy

Strong answers usually:

- define the concept simply
- mention why it matters
- compare it with a related concept when helpful
- give a short example

Use the example files and exercises here like mock interview practice.
