# XSLT and BaseX Cheat Sheet

- XSLT root template often matches `/`
- `xsl:apply-templates` keeps stylesheets modular
- `xsl:value-of` outputs simple values
- BaseX can run XPath, XQuery, and updates
- BaseX GUI is great for browsing trees
- BaseX CLI is great for repeatable workflows
- Namespace-aware queries need explicit prefixes in query context
