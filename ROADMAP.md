# XML Learning Roadmap

This roadmap gives you a practical eight-week path through the repository.

## Week 1: Orientation and Syntax

- Read [`00-introduction`](./00-introduction/README.md)
- Complete all exercises in [`01-xml-basics`](./01-xml-basics/README.md)
- Goal: write well-formed XML confidently
- Milestone: create a small XML profile document from scratch

## Week 2: Validation Foundations

- Study [`02-validation-dtd-xsd`](./02-validation-dtd-xsd/README.md)
- Compare DTD and XSD using the example documents
- Goal: understand the difference between well-formed and valid XML
- Milestone: validate a book catalog document with both DTD and XSD

## Week 3: Namespaces and Structured Documents

- Finish [`03-namespaces`](./03-namespaces/README.md)
- Review cross-links from namespaces back to validation
- Goal: read and write namespace-aware documents without confusion
- Milestone: merge two vocabularies into one XML document safely

## Week 4: XPath Practice

- Work through [`04-xpath`](./04-xpath/README.md)
- Use the university and e-commerce datasets for query practice
- Goal: navigate, filter, and extract XML data quickly
- Milestone: solve the hard XPath exercises without reading the solutions first

## Week 5: XQuery Reporting

- Complete [`05-xquery`](./05-xquery/README.md)
- Build small reports from project datasets
- Goal: write FLWOR expressions and shape clean XML output
- Milestone: create a department summary or discount report

## Week 6: XSLT Transformation

- Study [`06-xslt-and-transformation`](./06-xslt-and-transformation/README.md)
- Transform XML into HTML and report XML
- Goal: understand template matching and reusable transformations
- Milestone: generate an HTML page from XML data

## Week 7: Modeling and BaseX

- Read [`07-xml-modeling-and-best-practices`](./07-xml-modeling-and-best-practices/README.md)
- Complete [`08-basex`](./08-basex/README.md)
- Goal: design XML intentionally and query it in a database-oriented workflow
- Milestone: load a project dataset into BaseX and run query/update commands

## Week 8: Capstones and Interview Prep

- Build the three projects in [`09-real-world-projects`](./09-real-world-projects/README.md)
- Revise with [`10-cheat-sheets`](./10-cheat-sheets/README.md)
- Practice questions in [`11-interview-prep`](./11-interview-prep/README.md)
- Goal: move from theory to portfolio-ready XML work
- Milestone: complete at least one guided capstone and one open-ended extension

## Recommended Checkpoints

- End of Week 2: you can explain well-formed vs valid XML clearly
- End of Week 4: you can answer XPath questions on realistic datasets
- End of Week 6: you can produce HTML or summary XML from source XML
- End of Week 8: you can discuss modeling, validation, querying, transformation, and BaseX in interviews
