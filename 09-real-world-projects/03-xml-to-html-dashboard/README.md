# Project 03: XML to HTML Dashboard

This project focuses on KPI reporting and transformation. The dataset uses a default namespace so you can practice namespace-aware querying and XSLT.

## What You Will Practice

- working with namespaced XML
- creating compact XQuery reports from KPI data
- transforming XML into an HTML dashboard
- using the same data in BaseX with namespace-aware queries

## Project Files

- `data/dashboard.xml`
- `schemas/dashboard.xsd`
- `queries/`
- `transformations/`
- `exercises/`
- `solutions/`
- `expected-output/`

## Suggested Workflow

1. Inspect the namespaced XML in `data/dashboard.xml`
2. Review the XSD and note the target namespace
3. Run the sample XQuery with an explicit namespace declaration
4. Study the HTML transformation
5. Complete the guided and open-ended exercises
