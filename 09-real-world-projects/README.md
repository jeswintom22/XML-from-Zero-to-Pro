# 09 Real-World Projects

This chapter turns the earlier lessons into portfolio-style XML work.

## Learning Goals

- Apply XML syntax, validation, querying, and transformation together
- Work with realistic datasets instead of toy snippets
- Practice end-to-end workflows using schemas, XPath, XQuery, and XSLT
- Build confidence before moving into revision and interview prep

## How To Work Through The Projects

For each project:

1. Read the project `README.md`
2. Inspect the XML data in `data/`
3. Review the schema files in `schemas/`
4. Run or read the sample queries in `queries/`
5. Study the transformations in `transformations/`
6. Solve the guided and open-ended exercises

## Suggested Order

- Start with the university project for broad XML structure practice
- Move to e-commerce for product modeling and reporting
- Finish with the dashboard project for XML-to-HTML transformation practice
