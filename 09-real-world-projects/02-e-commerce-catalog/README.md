# Project 02: E-commerce Catalog

This project models a realistic product catalog with categories, brands, products, pricing, inventory, variants, and reviews.

## What You Will Practice

- product-oriented XML modeling
- XSD-based validation
- XPath and XQuery reporting over pricing and stock data
- XSLT transformation into a storefront-style HTML view
- BaseX queries over product data

## Project Files

- `data/catalog.xml`
- `schemas/catalog.xsd`
- `queries/`
- `transformations/`
- `exercises/`
- `solutions/`
- `expected-output/`

## Suggested Workflow

1. Inspect the XML structure in `data/catalog.xml`
2. Review the XSD to understand constraints
3. Run the discount and inventory reports
4. Study the HTML transformation
5. Complete the guided and open-ended tasks

## Good Practice Questions

- Which products are discounted?
- Which electronics items are in stock?
- Which brand has the most active products?
- Which product variants are nearly out of stock?
