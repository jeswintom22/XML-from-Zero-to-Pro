# Project 01: University Information System

This capstone models a university domain with departments, instructors, courses, students, rooms, schedules, and enrollments.

## What You Will Practice

- reading a larger XML dataset
- validating the data structure with DTD and XSD
- writing XPath and XQuery against linked entities
- generating summary output with XSLT
- loading the dataset into BaseX

## Project Files

- `data/university.xml`
- `schemas/university.dtd`
- `schemas/university.xsd`
- `queries/`
- `transformations/`
- `exercises/`
- `solutions/`
- `expected-output/`

## Suggested Workflow

1. Inspect the XML tree in `data/university.xml`
2. Review `schemas/university.dtd` and `schemas/university.xsd`
3. Run the sample queries
4. Review the sample XSLT
5. Complete the guided task before attempting the open-ended task

## Good Practice Questions

- Which courses belong to each department?
- Which students are enrolled in a given course?
- Which rooms have the highest capacity?
- How many courses does each instructor teach?
